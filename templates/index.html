RAINFALL PREDICTION SYSTEM

A place that help users to the predict the rainfall flow based on the given input data.





Features

Predicting Rainfall

Our system analyses the data from the given input and predicts whether there will be a rain tomorrow or not. And also using the data of average flow of rainfall of four months the average flow of rainfall for the next consecutive month can be c alculated.

Chart based output

From the given data it analyses and produces Visual Representation using Chart Studio which is a sub-product of Plotly

Download Predicted Data

Our System lets users download the predicted csv file, if user provides the data through csv. Users can also download the plots which is provided by Plotly.
